## 🚀 About The Project

**Notes CSBS** is a lightweight, secure, and student-focused academic resource platform designed specifically for **Computer Science & Business Systems (CSBS)** students at BMS College of Engineering, Bangalore.

We prioritize **simplicity**, **accessibility**, and **performance**, ensuring that students can access study materials quickly, even on lower-end devices or slow networks.

### ✨ Key Features

| Feature                    | Description                                                                            |
| :------------------------- | :------------------------------------------------------------------------------------- |
| **⚡ Blazing Fast**        | Optimized with Vite and lazy loading for instant access on any device via **PWA**.     |
| **📱 Mobile First**        | Fully responsive design that works perfectly on phones, tablets, and desktops.         |
| **🌙 Dark Mode**           | Eye-friendly dark theme enabled by default for late-night study sessions.              |
| **🔒 Secure & Safe**       | Built with modern security standards (CSP headers, secure API) to keep your data safe. |
| **📝 Organized Resources** | Intuitive access to Notes, Question Banks, PYQs (CIE/SEE), and Lab Manuals.            |
| **🔍 Smart Search**        | Quickly find subjects and resources without hassle.                                    |

## 🛠️ Technology Stack

This project leverages a modern, robust stack for maximum performance and safety:

- **Frontend**: React 18, TypeScript, Tailwind CSS, Shadcn UI
- **Backend & Database**: Supabase (PostgreSQL, Auth, Storage)
- **Tooling**: Vite (Build tool), ESLint (Quality), PWA (Offline support)
- **Deployment**: Vercel / Netlify (Recommended)

## 📦 Getting Started

Follow these simple steps to run the project locally on your machine.

### Prerequisites

- **Node.js** (v18 or higher) - [Download here](https://nodejs.org/)
- **npm** or **yarn**
- **Git**

### 🔧 Installation Guide

1. **Clone the repository**

   ```bash
   git clone https://github.com/notescsbsbmsce/notescsbs.git
   cd notescsbs
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env` file in the root directory (safe and secure):

   ```env
   VITE_SUPABASE_URL=your_project_url
   VITE_SUPABASE_ANON_KEY=your_anon_key
   ```

4. **Run the Development Server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

## 🛡️ Safety & Security

We take security seriously:

- **Environment Variables**: API keys are kept in `.env` files and never committed to Git.
- **Content Security Policy**: Implemented strict CSP headers to prevent XSS attacks.
- **Data Protection**: Supabase handles data securely with Row Level Security (RLS).
- **No Tracking**: We respect user privacy and do not extensive track personal data.
